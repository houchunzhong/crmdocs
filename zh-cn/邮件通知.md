# **邮件与通知模块使用说明书**  
**My Investment Markets（MIM）CRM 系统**

本手册详细说明 **MIM CRM** 中“邮件与通知”模块的功能、配置方法及操作流程，适用于市场、客服及运营团队，用于高效管理面向用户、代理及内部人员的多语言通信。

---

## **一、模块概览**

“邮件与通知”模块是 MIM CRM 的核心沟通工具，支持两类主要功能：

- ✅ **邮件列表**：创建、编辑并发送批量电子邮件，支持多语言版本与精准受众分发  
- ✅ **通知列表**：配置系统内通知（如登录弹窗、站内信），用于发布公告、政策更新或营销活动  

> 🔐 仅授权人员（如管理员、市场主管）可创建或修改内容，所有操作均保留审计日志。

## **三、核心功能详解**

### **1. 邮件列表（Email List）**

#### **功能描述**  
用于创建和发送批量营销或服务类电子邮件，支持多语言、指定收件人及发送效果追踪。

#### **字段说明**
| 字段 | 说明 |
|------|------|
| **ID** | 邮件唯一编号（如 `59`） |
| **邮件标题** | 邮件主题（如“年终奖金活动开启！”） |
| **发送状态** | 当前状态：<br>🟢 `发送成功`<br>🟡 `草稿`<br>🔴 `发送失败` |
| **发送范围** | 收件人群体：<br>`指定收件人`（自定义名单）<br>`全局`（所有注册用户） |
| **投递成功 / 收件人数** | 成功送达数量 vs. 总接收人数（如 `1/2`） |
| **默认语言** | 默认展示语言（如 英语、中文、印尼语） |
| **语言数量** | 支持的语言版本数（如 3 种） |
| **最后编辑时间** | 最近一次修改时间 |
| **发送时间** | 实际发送时间（若未发送则为 `-`） |
| **操作** | 按钮：**查看**（进入详情页） |

#### **操作流程**
1. 点击顶部 **“新邮件”** 按钮。
2. 填写邮件信息：
   - 标题、正文内容（支持富文本）
   - 上传附件（如 PDF、图片）
3. 配置多语言版本（可选）：
   - 添加英文、日文、越南语等翻译
   - 系统将根据用户语言偏好自动匹配
4. 设置发送范围：
   - 选择“全局”或“指定收件人”
   - 若指定，可手动输入用户 ID 或上传 CSV 名单
5. 预览邮件效果，保存为草稿或直接点击 **“发送”**。
6. 发送后，系统自动记录状态，并在列表中显示投递成功率。

> 💡 建议：对重要邮件先进行“测试发送”，验证链接、排版及翻译准确性。

---

### **2. 通知列表（Notification List）**

#### **功能描述**  
用于配置系统内通知，用户登录后将在页面顶部或弹窗中看到重要信息，如合规提醒、节日祝福或平台维护公告。

#### **字段说明**
| 字段 | 说明 |
|------|------|
| **ID** | 通知唯一编号（如 `28`） |
| **通知标题** | 简洁明了的标题（如“账户安全升级”） |
| **通知状态** | 内容状态：<br>🟢 `正式`（已完成）<br>🟡 `草稿`（未完成） |
| **发送开关** | 控制是否向用户展示：<br>✅ 开启 → 用户可见<br>❌ 关闭 → 隐藏 |
| **发送范围** | 受众群体：<br>`指定收件人`（特定用户）<br>`全局`（所有用户） |
| **已读 / 全部人数** | 已阅读人数 vs. 总接收人数（如 `1/1`） |
| **默认语言** | 默认显示语言 |
| **语言数量** | 支持的语言版本数 |
| **最后编辑时间** | 最近一次修改时间 |
| **发送时间** | 通知发布时间 |
| **操作** | 按钮：**查看**（查看详情） |

#### **操作流程**
1. 点击顶部 **“新通知”** 按钮。
2. 输入通知标题与正文（建议控制在 100 字以内）。
3. 配置多语言版本（可选）。
4. 选择发送范围（全局或指定用户）。
5. 启用 **“发送开关”** 以激活通知。
6. 保存后，符合条件的用户在下次登录时将看到该通知。
7. 可随时返回关闭开关，停止通知展示。

> 📌 注意：通知内容应简洁、紧急且必要，避免频繁打扰用户。

## **四、多语言支持**

- 支持多种语言（如中文、英文、日语、印尼语、越南语、泰语等）。
- 每条邮件或通知可配置多个语言版本。
- 系统根据用户的注册语言或个人设置自动匹配最合适的版本。

---

## **五、审计与追踪**

- 所有邮件与通知均保留完整历史记录。
- 包含：
  - 创建人、编辑时间
  - 发送/发布时间
  - 接收人列表
  - 投递/阅读状态
- 支持导出为 Excel，用于效果分析或合规审查。

## **七、常见问题解答（FAQ）**

### **Q1：如何向 VIP 客户群发送专属邮件？**  
- 在“发送范围”中选择“指定收件人”，然后导入 VIP 用户 ID 列表（CSV 格式）。

### **Q2：邮件发送失败的原因有哪些？**  
- 邮箱地址无效或格式错误  
- 邮件服务器连接异常  
- 内容触发反垃圾规则（如过多链接、敏感词）  
- 可通过“发送状态”中的错误日志排查

### **Q3：通知可以设置只显示一次吗？**  
- ✅ 是的。系统默认通知仅在用户首次登录时显示一次，阅读后不再重复弹出。

### **Q4：能否撤回已发送的邮件？**  
- ❌ 不支持撤回。但可立即关闭相关通知，或发送更正邮件说明。
