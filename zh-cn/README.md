# **CRM 文档阅读顺序与结构指南**

## **1. 文档整体结构概览**

MIM CRM 官方文档采用 **"模块化 + 层级导航"** 结构，分为五大类别：

| 类别                         | 包含模块                                   | 适用角色                   |
| ---------------------------- | ------------------------------------------ | -------------------------- |
| **1. 快速开始**              | 准备工作、授权与登录                       | 所有用户                   |
| **2. 账号配置**              | 账号类型设置、交易服务器概览、历史订单     | 管理员、IB                 |
| **3. 代理管理**              | 代理列表、代理申请记录、佣金设置           | IB、财务、管理员           |
| **4. 用户管理**              | 用户查询、用户列表、用户关系树、KYC 审核、资金管理 | 管理员、合规官、财务   |
| **5. 数据分析与报表**        | 资金数据、渠道管理、流量与转化分析         | 高级管理层、运营团队       |

> 📌 推荐阅读顺序：从左到右，从基础到高级，从通用到专项。

---

## **2. 推荐阅读顺序与学习路径**

### **阶段一：入门准备（面向所有用户）**

> ✅ 目标：了解基本概念与登录流程

1. **快速开始**
   - 1.1 准备工作：确认设备环境、浏览器兼容性、网络连接
   - 1.2 授权与登录：学习通过 OAuth 或 API 访问系统
   - 🎯 成果：成功登录并访问仪表盘

> 🔍 提示：这是所有后续操作的基础——务必首先完成。

---

### **阶段二：基础配置与代理管理（面向 IB 和管理员）**

> ✅ 目标：掌握账户设置与代理协作机制

2. **账号配置**

   - 2.1 账号类型设置：了解不同账户类型（真实账户 vs 模拟账户）
   - 2.2 交易服务器概览：查看可用交易节点与延迟
   - 2.3 历史订单：学习导出与分析历史交易记录
   - 🎯 成果：能够向客户推荐合适的账户类型

3. **代理管理**

   - 3.1 代理列表：查看所有已注册 IB 及其业绩
   - 3.2 代理申请记录：处理新的 IB 注册请求
   - 3.3 佣金设置：配置与监控 IB 佣金规则
   - 🎯 成果：有效管理与激励代理团队

> 💡 建议：IB 和运营人员应优先阅读此部分。

---

### **阶段三：核心用户与资金管理（面向管理员、合规、财务）**

> ✅ 目标：全面掌握用户生命周期与资金流转

4. **用户管理**
   - 4.1 用户查询 / 用户列表：快速定位目标客户
   - 4.2 用户关系树：可视化用户与 IB 之间的层级关系
   - 4.3 KYC 审核：
     - 待审核用户 → 全部用户 → 待审核提款账户 → 全部提款账户
     - 深入理解 KYC 流程、证件验证、状态流转
   - 4.4 资金管理：
     - 待办入金 / 入金记录 → 待办出金 / 出金记录 → 转账记录 → 退款记录
     - 学习如何安全合规地处理存取款请求
   - 🎯 成果：完全掌握用户全生命周期管理

> ⚠️ 注意：此为核心业务模块——请分配充足时间学习。

---

### **阶段四：数据分析与决策支持（面向高级管理层与运营）**

> ✅ 目标：以数据驱动策略决策

5. **数据分析与报表**
   - 5.1 资金数据：分析资金流入/流出趋势
   - 5.2 渠道管理：评估各渠道获客效率
   - 5.3 流量与转化分析：跟踪用户从注册到交易的完整旅程
   - 🎯 成果：生成报表指导营销与产品优化

> 📈 建议：结合 Excel 或 BI 工具进行深度分析。

---

### **阶段五：个人中心与系统维护（面向所有用户）**

> ✅ 目标：管理个人设置与系统日志

6. **个人中心**

   - 我的资料：更新联系方式、密码、通知偏好
   - 货币与支付：绑定支付方式、设置默认货币
   - 支付通道：查看支持的银行、钱包、第三方网关

7. **管理员面板**

   - 管理员列表：查看权限分配情况
   - 系统日志：审计关键操作、排查异常

> 🔐 提示：管理员应定期检查日志以确保安全。

## **5. 常见问题（FAQ）**

### **Q1：我应该先读哪个模块？**

- ✅ 新用户 → **快速开始**
- IB/代理 → **代理管理**
- 管理员 → **用户管理 + 资金管理**
- 高管 → **数据分析与报表**

---

### **Q2：是否需要阅读每个模块？**

- ❌ 不需要。根据角色选择性阅读：
  - 财务人员重点关注"资金管理"和"全部提款账户"
  - 合规人员重点关注"KYC 审核"

---

### **Q3：如何快速找到特定功能？**

- ✅ 使用左侧导航栏搜索关键词（如"KYC"、"出金"、"佣金"）
- 或使用 Ctrl+F 在文档内搜索

---

### **Q4：文档会更新吗？**

- ✅ 会的。每次系统版本发布后会同步更新，请关注"最后更新"时间戳。
