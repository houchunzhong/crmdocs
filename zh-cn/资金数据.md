# **资金数据模块使用说明书**

## **一、模块概览**

“资金数据”模块是 MIM CRM 的**财务分析中枢**，通过结构化表格呈现每日资金动态，支持以下功能：

- ✅ **日度资金报表**：查看每日新注册、活跃用户、入金/出金金额与人数
- ✅ **关键比率计算**：自动计算“出金入金比”、“成功出金率”等核心指标
- ✅ **趋势洞察**：横向对比多日数据，识别增长或风险信号

> 📊 所有数据基于系统实时统计生成，支持导出 CSV 用于深度分析。

## **三、核心功能详解**

### **1. 资金数据列表（Daily Funding Report）**

#### **功能描述**

以表格形式展示过去若干天的资金流动情况，每行代表一天的数据，便于横向对比和趋势分析。

#### **字段说明**

| 字段                   | 说明                                          |
| ---------------------- | --------------------------------------------- |
| **日期**         | 数据所属日期（格式：YYYY/MM/DD）              |
| **新注册**       | 当日新增用户数量                              |
| **活跃用户**     | 当日登录或交易的用户数                        |
| **入金金额**     | 当日所有成功入金总额（单位：USD）             |
| **入金次数**     | 当日入金操作总次数                            |
| **入金人数**     | 当日完成至少一次入金的用户数                  |
| **新增入金**     | 当日首次入金的用户所贡献的金额                |
| **留存入金**     | 非首次入金用户的累计金额                      |
| **申请出金**     | 当日提交的出金申请总额                        |
| **出金次数**     | 出金申请总次数                                |
| **出金人数**     | 提交出金申请的用户数                          |
| **成功出金金额** | 已处理成功的出金总额                          |
| **成功出金次数** | 成功出金的操作次数                            |
| **成功出金人数** | 成功完成出金的用户数                          |
| **出金入金比**   | `成功出金金额 / 入金金额`，反映资金流出压力 |

#### **示例解读**

假设某日数据如下：

```
日期：2025/10/27
入金金额：78,900 USD
成功出金金额：12,000 USD
出金入金比：15.21%
```

→ 表明当日资金流入强劲，流出比例较低，平台资金净流入状态良好。

> 💡 小贴士：若连续多日“出金入金比”超过 30%，需警惕客户流失或提现需求激增。

### **2. 关键指标含义与应用场景**

| 指标                           | 含义           | 应用场景               |
| ------------------------------ | -------------- | ---------------------- |
| **新注册**               | 用户增长速度   | 判断市场推广效果       |
| **活跃用户**             | 用户粘性       | 评估产品吸引力         |
| **入金金额**             | 资金注入强度   | 监控业务收入来源       |
| **新增入金**             | 新用户贡献     | 评估获客质量           |
| **留存入金**             | 老用户复购能力 | 判断用户生命周期价值   |
| **申请出金 vs 成功出金** | 处理效率       | 发现支付延迟或技术问题 |
| **出金入金比**           | 资金流动性风险 | 预警潜在现金流压力     |

## **四、最佳实践建议**

| 场景               | 建议                                                       |
| ------------------ | ---------------------------------------------------------- |
| **日常监控** | 每日查看“入金金额”与“出金入金比”，建立基线             |

## **五、常见问题解答（FAQ）**

### **Q1：数据更新频率是多少？**

- ✅ 实时同步。每次交易完成后立即计入对应日期。

### **Q2：为什么“成功出金金额”小于“申请出金”？**

- ❌ 正常现象。部分出金可能因审核未通过、账户余额不足等原因失败。

### **Q3：能否导出完整数据？**

- ✅ 可以。点击右上角 **“导出 CSV”** 按钮，下载全部记录（含隐藏列）。

### **Q4：如何查看特定时间段的数据？**

- ⚠️ 当前版本仅支持最近 15 天。未来将支持自定义日期范围筛选。

## **六、总结**

MIM CRM 的“资金数据”模块为企业提供了透明、精准的资金流动视图。通过结构化的日度报告和关键比率分析，管理员能够快速掌握业务动态，及时响应市场变化，优化资金管理策略。

✅ **推荐操作流程**：

1. 每日查看“入金金额”与“出金入金比” → 2. 定期分析“新增入金”趋势 → 3. 对比“活跃用户”与“新注册” → 4. 识别异常并采取行动

> 📌 所有数据均来自真实交易记录，确保决策依据可靠。
