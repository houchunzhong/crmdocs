# **渠道管理**  

## **一、模块概览**

“货币与支付”模块是 MIM CRM 的**金融中枢**，包含两大核心子系统：

- ✅ **币种管理**：定义支持的法币与数字货币，配置显示格式，并设置兑换汇率  
- ✅ **支付通道管理**：集成并配置入金/出金方式，包括银行转账、第三方支付及加密钱包  

> 🔐 仅限**系统管理员**或**财务主管**访问。所有变更均记录于审计日志，确保操作可追溯。

## **三、核心功能详解**

### **1. 币种管理**

#### **功能描述**  
添加、编辑或删除支持的币种（包括法币如 USD、PHP 和数字货币如 USDT、BTC）。每个币种可独立配置：
- 显示格式（小数位数、千分位分隔符）
- 定价方式（手动 / 自动）
- 兑换汇率（用于入金与出金）

#### **字段说明**
| 字段 | 说明 |
|------|------|
| **币种代码** | 系统内部唯一标识（如 `USDT`） |
| **币种名称** | 用户端显示名称（如 “泰达币”） |
| **币种类型** | `数字货币` 或 `法币` |
| **小数位数** | 小数点后显示位数（如 2 → $1.00） |
| **定价方式** | `手动`（管理员设定）或 `自动`（通过实时 API 获取） |
| **兑换对** | 基准报价对（如 `USDT/USD`） |
| **入金汇率** | 用户入金时使用的汇率（如 1 USDT = 1.00003 USD） |
| **出金汇率** | 用户出金时使用的汇率（如 1 USDT = 0.99998 USD） |
| **编辑** | 修改当前币种设置 |

#### **操作流程**
1. 点击 **“新增币种”**。
2. 填写币种信息：
   - 选择 **币种代码** 与 **名称**
   - 选择 **类型**（数字货币 / 法币）
   - 设置 **显示格式**（小数位、分隔符）
   - 选择 **定价方式**（手动/自动）
   - 定义 **兑换对**（如 USDT/USD）
   - 输入 **入金汇率** 与 **出金汇率**
3. 点击 **“保存”** 完成配置。
4. 新币种将出现在列表中，并可在支付通道中启用。

> 💡 示例：若需支持 USDC，需先在此创建币种条目，并定义其与 USD 的兑换关系。

### **2. 支付通道管理**

#### **功能描述**  
配置并管理所有可用的资金通道，包括银行电汇、第三方支付（如 Skrill）及加密钱包（如 TRON、ERC20）。每个通道可独立设置：
- 支持币种
- 费率结构
- 交易限额
- 启用状态

#### **字段说明**
| 字段 | 说明 |
|------|------|
| **ID** | 通道唯一编号 |
| **显示名称** | 用户端可见名称（如 “银行转账”） |
| **排序优先级** | 显示顺序（数值越小越靠前） |
| **状态** | `启用` / `停用` |
| **方向
