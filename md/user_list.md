# **User Management Module Manual**

The User Management section of the **My Investment Markets** CRM enables administrators to manage client profiles, monitor registration data, and visualize the hierarchical relationships between investors and partners.

---

### **1. User List Module**

The **User List** serves as the central database for all registered clients, offering a comprehensive tabular view of account details and administrative actions.

#### **Key Features & Navigation**

* **Advanced Filtering:** Filter users by "Last CRM Login" or "Registration" date ranges.
* **Demographic Segmentation:** Use dropdown menus to filter by Nationality and User Level.
* **Global Search:** A multi-field search bar allows quick lookup by User ID, Name, Mobile Number, or Email.
* **Data Export:** The **"Download to Excel"** button exports the current filtered dataset—or the full list—for offline reporting.
* **Status Indicators:**
  * **Verification Status:** Green checkmarks indicate verified accounts; orange alert icons highlight pending or flagged statuses.
  * **Contact Verification:** Icons next to phone numbers and email addresses show whether each contact method has been verified.
* **Pagination Control:** Navigate large datasets using page controls or adjust “Records per page” (from 10 to 100) in the bottom-right corner.

---

### **2. User Relationship Tree Module**

The **User Relationship Tree** provides an interactive, nested visualization of user connections—commonly used to map referral networks and sub-agent hierarchies.

#### **Key Features & Navigation**

* **Hierarchical Structure:** An expandable/collapsible tree layout uses **[+]** and **[-]** icons to reveal or hide "Parent" and "Child" account relationships.
* **Real-Time Financial Metrics:** Each node displays live financial data:
  * **Account Balance (Green):** Shows the user’s current equity or available balance.
  * **Trading Exposure (Red):** Reflects total trading volume or outstanding exposure.
* **Geographical Context:** National flags next to usernames offer instant visual identification of regional distribution within the network.
* **Portfolio Identification:** A briefcase icon indicates the user’s account or portfolio type.
* **Dedicated Tree Search:** A specialized search bar at the top lets you jump directly to a specific user within a large organizational structure.

| **Feature**         | **User List**                          | **User Relationship Tree**            |
|---------------------|----------------------------------------|---------------------------------------|
| **Primary Goal**    | General administration and data filtering. | Tracking referral chains and hierarchical structures. |
| **Key Metrics**     | Contact info, verification status, and registration dates. | Account balances, exposure, and parent-child links. |
| **Data Format**     | Linear table view.                     | Nested tree / diagram view.           |
